## Installation
Follow the steps below to set up the Laravel application:
1. Add to composer.json:
``` 
"repositories": [
    {
        "type": "vcs",
        "url": "https://github.com/operatorx1/tool-laravel.git"
    }
],
"require": {
  "operatorx1/tool_laravel": "dev-main"
}

```

2. Update package:
```
composer update;
```
